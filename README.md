# Portfolio OS — Experiencia de Escritorio Web

Este proyecto es una interfaz de usuario interactiva que simula un sistema operativo web, diseñada para mostrar mis habilidades como **Ingeniero en Informática Full Stack** con un enfoque profundo en **Backend y Arquitectura de Software**. 

A través de esta plataforma interactiva, presento mi filosofía de desarrollo: crear interfaces de vanguardia mientras aplico una planificación minuciosa y una atención impecable al detalle técnico.

## 🚀 Filosofía y Principios de Ingeniería

El desarrollo de este portafolio refleja los principios técnicos que aplico a sistemas en producción real:

- **Planificación y Estructura:** Antes de escribir código, se prioriza el diseño arquitectónico y el modelo de datos. Un código estructurado es un código predecible y escalable.
- **Robustez y Seguridad:** Aplicación de mecanismos avanzados de control (ej. bloqueos atómicos en Redis y gestión de concurrencia) para mantener la integridad de los datos en escenarios críticos.
- **Eficiencia Algorítmica:** Optimización profunda de recursos (como en implementaciones FIFO para inventarios y optimización de consultas complejas en BD).
- **Estética de Vanguardia:** Creación de interfaces altamente refinadas (UI/UX) que aseguran una experiencia visual responsiva, intuitiva y premium sin comprometer el rendimiento.

## 🛠 Stack Tecnológico

Este entorno virtual está construido con:
- **Frontend Core:** React 19, Vite, y TailwindCSS v4.
- **Animaciones y UX:** Framer Motion para transiciones fluidas de ventanas y secuencias de inicio.
- **Manejo de Estado:** Context API estructurado para la persistencia del gestor de ventanas (`WindowContext`).
- **Iconografía:** Lucide React y SVG nativos.

## 🌟 Características Clave de la Interfaz

- **Gestor de Ventanas (Window Manager):** Soporte completo para arrastrar, minimizar, maximizar y apagar aplicaciones con control de z-index dinámico.
- **Soporte de Tema (Light/Dark Mode):** Sincronización impecable del modo claro y oscuro, afectando al 100% de la interfaz (desde fondos dinámicos de alta calidad hasta contenedores interiores).
- **Secuencia de Encendido (Boot/BIOS):** Simulador de carga de núcleo del sistema y autenticación al interactuar con el botón de "Apagar" o al ingresar a la plataforma.
- **Aplicaciones Integradas:** Navegador interno independiente y panel de contacto con accesos optimizados a perfiles oficiales y comunicación directa.

## 💻 Entorno Local

Para clonar y ejecutar este entorno de desarrollo en tu máquina:

```bash
# 1. Clonar el repositorio
git clone <tu-repositorio-url>
cd portfolio-os

# 2. Instalar dependencias
npm install

# 3. Levantar el servidor de desarrollo local
npm run dev
```
